# 向精神薬チェックツール

## 目的

在庫データを取り込み、対象商品のチェック用資料を作成するアプリです。

元データをアップロードすると、商品マスタと照合して、次の形のExcelを出力します。

| 商品コード | 商品名 | 包装単位 | 商品マスタ.記号 | 数 |
|---|---|---|---|---|

## タブ構成

### 1. 在庫資料作成

通常作業者向けの画面です（通常はこのタブのみ使用）。

Step1の操作画像を説明文つきで順番に確認しながら在庫データを保存し、Step2で保存したファイルをドラッグ＆ドロップ、または Upload ボタンからアップロードして在庫資料Excelを作成します。


商品マスタは `masters/product_master.csv`（または管理者用タブで更新した内容）を自動使用し、通常作業者が指定する項目はありません。

### 2. 管理者用

商品マスタ更新時だけ使う画面です（通常作業者は使用しません）。

整形前の商品一覧Excel/CSVから、対象の記号だけを抽出して、アプリ用の `product_master.csv` を作成します。

管理者用タブで「このマスタで更新する」を押すと、アプリ内の `masters/product_master.csv` を上書きし、すぐに在庫資料作成タブで使用できます。

## 商品マスタCSV

`masters/product_master.csv` を置く場合は、最低限この列が必要です。

```csv
商品コード,記号
a058565032,向精神薬Ⅱ
a155197129,向精神薬Ⅲ
```

## 起動方法

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## 元データに必要な列

元データ側には、次の列が必要です。

- 商品コード
- 商品名
- 包装単位
- 数、数量、在庫数、現在庫数、在庫数量 のいずれか

## 出力シート名

```text
向精神クエリ
```

## 補足

在庫資料作成タブの Step1 / Step2 には、`guide` フォルダ内の画像を1枚ずつ表示し、`← 前の画像` / `次の画像 →` ボタンで切り替えるスライド形式の操作ガイドを表示します。

- Step1 用: `guide/step1_*.png`
- Step2 用: `guide/step2_*.png`

画像が存在するものだけを読み込みます（存在しない場合は表示しません）。

対象は、向精神薬Ⅱ・向精神薬Ⅲ・生活改善薬・毒薬を基本にしています。

並び順ロジックは後から追加する前提です。
現在は暫定で「商品マスタ.記号」「商品コード」順にしています。

CSVはUTF-8だけでなく、Excel由来のCP932/Shift-JISも読み込めるようにしています。
